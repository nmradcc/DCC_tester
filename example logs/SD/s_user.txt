User documenation summary for test software version 1.0.0

Summary of command line and 'SEND.CFG' switches:

Usage:     send [-?] [-u] [-m] [-a addr] [-d l|f|a|s] [-n pre] [-N trig]
                [-l] [-p port] [-f] [-x] [-r] [-t mask] [-c mask] [-E pre]
                [-T] [-F fill] [-R reps] [-P] [-A] [-s] [-S] [-g mask]
                [-o pair] [-k] [-D] [-e trg]

Cfg file: <SEND.CFG>

  -?                    Print usage message and exit
  -u                    Print user information to 's_user.txt' and exit
  -m         MANUAL     Start in manual mode                 <value false>
  -a <addr>  ADDRESS    Decoder address                      <value 3>
  -d l|f|a|s TYPE       Dec. type(l-LOC,f-FUNC,a-ACC,s-SIG)  <value l>
  -l         LAMP       Use rear lamp for function tests     <value false>
  -n <pre>   PRESET     Signal decoder preset aspect         <value 0>
  -N <trig>  TRIGGER    Signal decoder trigger aspect        <value 8>
  -p <port>  PORT       I/O Port                             <value 0x0340>
  -f         FRAGMENT   Test all fragments                   <value false>
  -x         CRITICAL   Protect critical regions             <value false>
  -r         REPEAT     Repeat decoder tests                 <value false>
  -t <mask>  TESTS      Bit mask of tests to run             <value 0x00000003>
  -c <mask>  CLOCKS     Bit mask of clocks to try            <value 0xfffffce7>
  -g <mask>  FUNCS      Bit mask of active functions         <value 0x1f>
  -E <pre>   EXTRA_PRE  Extra margin test preamble bits      <value 0>
  -T         TRIG_REV   Use loco reverse as trigger packet   <value false>
  -L         LOCO_FIRST Put loco packet before func packet   <value false>
  -F <fill>  FILL_MSEC  Fill time in milliseconds            <value 1000>
  -R <reps>  TEST_REPS  Non packet acceptance test repeats   <value 2>
  -P         LOG_PKTS   Send packets to log, not hardware    <value true>
  -A         NO_ABORT   Do not stop program on an error      <value false>
  -s         LATE_SCOPE Put scope trigger after trigger      <value false>
  -S    SAME_AMBIG_ADDR Use same address for ambig tests     <value false>
  -o        ACC_PAIR    Accessory output pair (1-4)          <value 1>
                            output {preset, trigger}         <value {0, 8}>
  -k        KICK_START  Kick start motor for function tests  <value false>
  -e <trg>   EXTRA_TRG  Extra Ames test trigger packets      <value 0>
  -D        DEBUG_ON    Log debug messages                   <value false>

Manual keyboard commands >

ESC - Return to command line       h - Print header
  c - Send single clock phase      C - Send series of clock phases
  u - Clear underflow              0 - Send zeros
  1 - Send ones                    a - Send scope A pattern
  b - Send scope B pattern         o - Send scope timing packet
  w - Send warble packets          S - Send stretched 0 pattern
  r - Send DCC reset packets       d - Send DCC packets
  D - Send stretched DCC packets   s - Change loco speed, acc. output
  e - Set speed to E-STOP          f - Change loco direction, acc. on/off
  E - Set speed to E_STOP(I)       t - Run self tests repeatedly
  k - Kickstart loco for funcs     i - Send DCC idle packets
  R - Send hard resets             g - Test generic I/O
  z - Run decoder tests            q - Quit program
